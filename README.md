#  Sales & Profit Dashboard in Excel

##  Project Overview

The **Sales & Profit Dashboard** is an interactive Excel dashboard designed to analyze and visualize business sales performance and profitability.  
This project transforms raw sales data into meaningful business insights using Pivot Tables, Pivot Charts, Slicers, and data visualization techniques in Microsoft Excel.

The dashboard helps users quickly understand:
- Sales performance
- Profit trends
- Customer analysis
- Regional performance
- Category-wise business insights

---
